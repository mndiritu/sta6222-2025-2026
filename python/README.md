# Python Code Structure – STA 6222

Python scripts and notebooks are organised by teaching week:

- `week01_intro/` – basic Python setup and toy simulations
- `week02_sir/`   – SIR model implementations
- `week03_seir/`  – SEIR model implementations and experiments

Use descriptive filenames, e.g.:

- `sir_basic_sim.py`
- `seir_intervention_scenarios.py`
