# STA 6222 – Models for Disease Transmission and Management
## 2025–2026 – R/Python Simulations

This repository contains R and Python code used for:
- Deterministic and stochastic epidemic models
- SIR/SEIR simulations
- Parameter exploration and visualisation
- In-class demos and student exercises

Structure:
- `R/` — R scripts and RMarkdown files
- `python/` — Python scripts and notebooks
- `data/` — example datasets (non-identifiable)
- `notes/` — helper notes and snippets
